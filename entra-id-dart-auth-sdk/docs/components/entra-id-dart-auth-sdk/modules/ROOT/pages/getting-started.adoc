= Getting Started
:page-layout: default

This page covers the quickest path to installing and navigating Entra ID Dart Auth SDK.

== Install the Package

Add the package to your pubspec.yaml:

[source,yaml]
----
dependencies:
  entra_id_dart_auth_sdk: ^0.0.7
----

Then run:

[source,bash]
----
dart pub get
----

== Import the Library

[source,dart]
----
import 'package:entra_id_dart_auth_sdk/entra_id_dart_auth_sdk.dart';
----

== Start with the Package Sources

* Pubspec: apps/sdks/entra-id-dart-auth-sdk/entra-id-dart-auth-sdk/pubspec.yaml
* README: apps/sdks/entra-id-dart-auth-sdk/entra-id-dart-auth-sdk/README.md
* Examples: apps/sdks/entra-id-dart-auth-sdk/entra-id-dart-auth-sdk/example
* Tests: apps/sdks/entra-id-dart-auth-sdk/entra-id-dart-auth-sdk/test

== Build the Docs

From apps/sdks/entra-id-dart-auth-sdk/entra-id-dart-auth-sdk/docs:

[source,bash]
----
npm install
npx antora antora-playbook.yml
----

The generated site is written to build/site.
